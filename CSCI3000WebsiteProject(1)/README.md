# Welcome!
## This website is the project of David and Gideon for our CSCI3000 Website Project
The goal is to create a website that will hold all of the information for Christian Karate Academy Federation, but will also include a working authentication form.

## Prototype 1 Goals
- Create the webpages and include the desired information
- Create basic CSS to begin to format the webpages
- Add photos into the mix
- Begin on the login form

## Prototype 2 Goals
- Rudimentary login processing
- JavaScript on all webpages for animation and usability
- Hidden pages only accessible with cookies from authentication
